## Getting Started

To get started with the project, follow these steps:

1. Ensure you have NodeJS 18 or newer installed on your system.
2. Install the necessary Nodejs packages using npm or yarn:

```bash
npm install
```
```bash
yarn add
```

3. Run the script:

```bash
node .
```
```bash
npm run start
```
